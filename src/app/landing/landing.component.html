🔒

100% Client-Side Pipeline

Your data never leaves your machine. There is no backend server storage or cloud processing.

1
Local File Input

Drag and drop DICOM or NIfTI directories directly into the browser viewport.

2
In-Browser Parsing

Files are read asynchronously in memory using JS/TS binary array streaming.

3
GPU-Accelerated Render

Textures are pushed straight to local VRAM for Three.js WebGL volume rendering.

🔒 Zero Server Footprint: Ideal for strict institutional firewalls and internal research sandbox protocols. Fully compliant with localized data protection workflows.

Automation

Intrinsic Spatial Alignment

Zero-Click Multiplanar Co-Registration

Forget manual registration step-ups. RADIWEB 3D automatically harmonizes heterogeneous matrix arrays on the fly.

  • Header Matrix Mapping: Automatically parses NIfTI affine transformations (sform/qform) to synchronize coordinates with DICOM physical space.
  • LPS / RAS Basis Transformation: Natively detects and re-orients coordinate spaces on the fly. Dynamically bridges DICOM native LPS (Left-Posterior-Superior) patient space with NIfTI research RAS (Right-Anterior-Superior) coordinate grids with zero directional inversion bugs.
  • Multimodal Overlay: Simultaneously view structural CT arrays, anatomical labels, and dose distributions perfectly aligned.
  • Resolution Agnostic: Seamless crosshair tracking across mismatched dimensions (e.g., mapping a 256x256 Atlas onto a 512x512 CT).
